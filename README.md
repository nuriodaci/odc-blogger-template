# ODC Blogger Template

Blogger theme files for the Wow Knowledge / Northstar Journal template.

## Files

- `inline.xml`: full inline Blogger theme source
- `theme.xml`: performance-oriented theme with critical inline CSS and external CSS/JS assets
- `style.css`: extracted theme styles
- `main.js`: extracted theme scripts

## Usage

1. Upload `theme.xml` to Blogger.
2. Keep `style.css` and `main.js` available from this repository CDN URL.
3. Use `inline.xml` as the full inline backup/source version.
