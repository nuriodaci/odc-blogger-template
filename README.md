# ODC Blogger Template

Blogger theme files for the Wow Knowledge theme.

## Preview

Live preview: https://wow-knowledge-com.blogspot.com/

## Files

- `theme.xml`: Blogger theme source with critical inline CSS and external CSS/JS assets
- `style.css`: extracted theme styles
- `main.js`: extracted theme scripts
- `desktop-ss.png`: desktop preview screenshot
- `mobile-ss.png`: mobile preview screenshot
- `pagespeed-mobile.png`: mobile PageSpeed reference screenshot

## Usage

1. Upload `theme.xml` to Blogger.
2. Keep `style.css` and `main.js` available from this repository CDN URL.
3. Use the preview site above to verify the published theme.
